# `@ssr-coin/vue`

Use `ssr-coin` with [Vue](https://github.com/vuejs/vue).

### Usage

Install `@ssr-coin/vue`.

~~~shell
$ npm install @ssr-coin/vue
~~~

The plugin is automatically loaded and
the `view` property of your page configs is now rendered with Vue.
