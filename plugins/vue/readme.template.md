# `@goldpage/vue`

Use Goldpage with [Vue](https://github.com/vuejs/vue).

### Usage

Install `@goldpage/vue`.

~~~shell
$ npm install @goldpage/vue
~~~

The plugin is automatically loaded and
the `view` property of your page configs is now rendered with Vue.
