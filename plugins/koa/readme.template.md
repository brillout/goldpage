# `@goldpage/koa`

Use `ssr-coin` with [Koa](https://github.com/koajs/koa).

### Usage

Install `@goldpage/koa`.

~~~shell
$ npm install @goldpage/koa
~~~

The plugin is automatically loaded and
a Koa middleware is now available at `require('ssr-coin').koa`.
