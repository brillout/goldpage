# `@ssr-coin/koa`

Use `ssr-coin` with [Koa](https://github.com/koajs/koa).

### Usage

Install `@ssr-coin/koa`.

~~~shell
$ npm install @ssr-coin/koa
~~~

The plugin is automatically loaded and
and a Koa middleware is now available at `require('ssr-coin').koa`.
