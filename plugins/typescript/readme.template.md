# `@goldpage/typescript`

Use `ssr-coin` with TypeScript.

### Usage

Install `@goldpage/typescript`.

~~~shell
$ npm install @goldpage/typescript
~~~

The `ssr-coin/typescript` plugin is automatically loaded.

