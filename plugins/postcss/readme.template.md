# `@goldpage/postcss`

Use `ssr-coin` with PostCSS.

### Usage

Install `@goldpage/postcss`.

~~~shell
$ npm install @goldpage/postcss
~~~

The `ssr-coin/postcss` plugin is automatically loaded.

Configure PostCSS:

~~~js
!INLINE ./example/ssr-coin.config.js --hide-source-path
~~~
