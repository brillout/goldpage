# `@goldpage/hapi`

Use Goldpage with [Hapi](https://github.com/hapijs/hapi).

### Usage

Install `@goldpage/hapi`.

~~~shell
$ npm install @goldpage/hapi
~~~

The plugin is automatically loaded and
a Hapi plugin is now available at `require('ssr-coin').hapi`.
