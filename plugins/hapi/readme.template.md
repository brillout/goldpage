# `@ssr-coin/hapi`

Use `ssr-coin` with [Hapi](https://github.com/hapijs/hapi).

### Usage

Install `@ssr-coin/hapi`.

~~~shell
$ npm install @ssr-coin/hapi
~~~

The plugin is automatically loaded and
and a Hapi plugin is now available at `require('ssr-coin').hapi`.
